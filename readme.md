# Cat-rousel

## Usage 

Install Deps

```
yarn
```

Start a server on :8080

```
yarn start
```

## Content

The project uses React + Redux.
It also leverages libs like Redux-thunk, Electron-CSS (my own CSS-in-JS lib) and Jest for unit testig

## Other

Run tests 

```
yarn test
```

Run lint

```
yarn lint
```
